Fliger is an Electron Based Launcher focused on Linux Desktops.
![Fliger_Unit_Convert](uploads/f4fa1f3f70dad12c9686d1eb05ececaf/Fliger_Unit_Convert.png)

There is Fliger where,
- Apps are **Listed**
- Math is **Calculated**
- Units are **Converted**
- Files are **Suggested**
- Images are **Shown**
- Videos are **Auto Played**
