Fliger is an Electron Based Launcher focused on Linux Desktops.
![Fliger_Logo](public/images/fliger.svg)

There is Fliger where,
- Apps are **Listed**
- Math is **Calculated**
- Units are **Converted**
- Files are **Suggested**
- Images are **Shown**
- Videos are **Auto Played**
